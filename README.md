# BeeSMART
Honning tappemaskine til den lille produktion.
WIFI baseret med brugerinterface på pc, tablet eller smartphone uden en app.

## Specs
Leverer inden for +/- 5g. Doseringsmængden kan indstilles mellem 100 og 750g.
